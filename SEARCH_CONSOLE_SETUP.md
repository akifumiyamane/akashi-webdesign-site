# Google Search Console Setup Guide

## Step 1: Add Property

1. Google Search Consoleを開く
2. 「プロパティを追加」から `ドメイン` を選択
3. 本番ドメインを入力

## Step 2: Verify Ownership

1. 表示されたTXTレコードをDNSへ追加
2. 反映後に「確認」を実行

## Step 3: Submit Sitemap

1. 左メニュー「サイトマップ」を開く
2. `sitemap.xml` を送信
3. ステータスが `成功しました` になることを確認

## Step 4: Request Indexing

1. URL検査でトップURLを入力
2. 「インデックス登録をリクエスト」を実行

## Step 5: Weekly Monitoring

- パフォーマンス: 表示回数 / クリック数 / CTR / 掲載順位
- 検索クエリ上位3件を記録
- CTRが低い場合は title/description を改善

## Profit Impact (Expected)

- 検索流入増: インデックス速度を上げ、早期流入を狙える。
- 商談機会増: クエリ分析から訴求改善し、問い合わせ率を高める。
